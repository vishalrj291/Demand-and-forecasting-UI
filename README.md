# 📊 Demand Forecasting System – Frontend (UI)

## 🚀 Overview

This repository contains the **frontend (UI)** of the Demand Forecasting System.
It is built using **HTML, CSS**, and focuses on providing a clean and responsive interface for users to interact with the forecasting model.

The UI is designed to collect input data and display predictions generated by the backend Machine Learning model.

---

## 🧠 Project Architecture

This project is part of a **full-stack ML system**:

* 🎨 Frontend (This Repo) → User Interface
* ⚙️ Backend (Python/ML Model) → Prediction Engine
* 🔗 API Communication → Connects UI with ML model

---

## ✨ Features

* Responsive UI design

## 🔌 Backend Integration

This UI is designed to work with a backend API.

Example API call:

```javascript
fetch("https://your-backend-url/predict", {
  method: "POST",
  headers: {
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    feature1: value1,
    feature2: value2
  })
})
.then(res => res.json())
.then(data => {
  console.log(data);
});
```

4. Deploy
